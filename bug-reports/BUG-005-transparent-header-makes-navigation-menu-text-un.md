## BUG-005 — Transparent header makes navigation menu text unreadable on light hero images

**Title:** Transparent header makes navigation menu text unreadable on light hero images
**Environment:** All browsers, Astra 4.6
**Severity:** Major | **Priority:** Medium

**Steps to Reproduce:**
1. Enable Transparent Header
2. Set hero section background to a white or very light image
3. View frontend

**Expected Result:** Text colour auto-adjusts for contrast, OR customizer provides separate text colour for transparent state
**Actual Result:** Menu items render in default light colour (white) — completely invisible against light hero; no transparent-state colour override available in free version
